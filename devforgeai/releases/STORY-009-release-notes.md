## Summary

This release adds daemon lifecycle management commands (`treelint daemon start|stop|status`), manual indexing command (`treelint index`), and auto-detection logic that automatically routes queries through the daemon if running or falls back to direct index queries.

---

## New Features

### Daemon Commands
- `treelint daemon start` - Start the background daemon process
- `treelint daemon stop` - Stop the running daemon
- `treelint daemon status` - Show daemon status (PID, uptime, indexed files/symbols)

### Index Command
- `treelint index` - Build or rebuild the symbol index
- `treelint index --force` - Force full re-index, bypassing hash cache
- Progress bar for large repositories (>10 files)

### Auto-Detection
- Search command automatically detects daemon availability
- Falls back to direct index query if daemon not running
- Builds index on-demand if no index exists
- Transparent to user (same output format regardless of path)
